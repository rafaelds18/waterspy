Para execução do codigo do projeto:
	*	Necessario ter o nodejs instalado, a versão 14.17.0
	*	Base Dados:
			*	Executar o ficheiro "CREATE_TABLES.sql", num editor SQL que suporte o SQL Server da Microsoft
	*	Backend:
			*	Para execução do Backend é necessario:
					*	Abrir a consola/terminal na pasta "\Backend\api\src"
					*	Executar o comando "npm install", para instalar todas a bibliotecas necessarias
					*	Executar o comando "npm run start", para executar o codigo
	*	Frontend:
			*	Para execução do Frontend é necessario:
					*	Abrir a consola/terminal na pasta "\Frontend\WaterSpy"
					*	Executar o comando "npm install", para instalar todas a bibliotecas necessarias
					*	Executar o comando "npm run start-dev", para executar o codigo
